THE ADVENTURES OF MR. TURTLE

This project was part of a final assignment for my computer science course. The project required that I create a fully functional and interactive program that covered multiple concepts used throughout the course. When presented with this challenge, I decided to first conduct as much research as I could on what type of program would work best to satisfy the project requirements. I decided to create a video game because not only would it incorporate all the features of the course, but it would make it a lot easier to create because of my passion for gaming. I then began the coding process in which I used the program called Processing to build my game. When designing this game, I found that the best approach was to break my code into smaller pieces to make it less overwhelming. For example, I decided to design my game elements first and then code each element afterwards. This allowed me to focus on just one piece of my program and make sure it was running properly before moving onto the next element. Upon completion of my game, I went to a couple of my friends and had them test the game to determine whether they liked it and if it needed improvements. They informed me that my game was really challenging and, so I went back into my code and reduced the speed of the enemies to make it more playable.

A skill that I was able to demonstrate in the production of this game was my problem-solving skills. There were many challenges that I faced when creating this game. For example, during the coding process of making the game, I ran into many problems where my code either didn't work or would disable other features of my game. I found out that the best way to fixing these problems was going back to review my code to understand what was triggering these certain responses. I would freeze certain sections of my code to test whether that area was working properly and move onto another piece of code until I found the problem. Overall, I believe I was able to successfully create an interactive video game for my final assignment.
