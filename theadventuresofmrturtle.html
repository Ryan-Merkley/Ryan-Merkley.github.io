THE ADVENTURES OF MR. TURTLE

This was a project that I worked on during my free time and outside of school. The reason why I decided to create a video game is because I had recently taken a computer science course that taught me how to code and I thought that it would be interesting to create a game for myself and family to play. I decided to create a remake of my favourite childhood game Frogger. I used to play Frogger all the time when I was growing up and thought that it would be a perfect opportunity to represent my game.

To build this game, I used the program called Processing and designed all my game elements throughout this program. When designing this game, I found that the best approach was to break my code into smaller pieces to make it less overwhelming. For example, I would design my game elements first and then code each element afterwards. This allowed me to focus on just one piece of my program to make sure it was running properly before moving onto the next element. Upon completion of my game, I went to a couple of my family members and had them test out my game to determine whether they liked it or not and if it needed any improvements. They informed me that my game was really challenging, so I went back into my code and reduced the speed of the enemies to make it more playable.

A skill that I was able to demonstrate in the production of this game was my problem-solving skills. There were many challenges that I faced when creating this game. For example, during the coding process of making the game, I ran into many problems where my code either didn't work or would disable other features of my game. I found out that the best way to fixing these problems was going back to review my code to understand what was triggering these certain responses. I would freeze certain sections of my code to test whether that area was working properly and move onto another piece of code until I found the problem. Overall, I believe I was able to successfully create an interactive video game for my personal usage as well as for my family to use.
